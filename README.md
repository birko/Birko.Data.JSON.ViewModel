# Birko.Data.JSON.ViewModel

JSON file-based ViewModel repository implementations for the Birko Framework.

## Features

- ViewModel repositories for JSON file storage
- Store validation support

## Installation

```bash
dotnet add package Birko.Data.JSON.ViewModel
```

## Dependencies

- Birko.Data
- Birko.Data.ViewModel
- Birko.Data.JSON
